# 《WebGL编程指南》学习笔记
WebGL权威指南typescript实现笔记,采用webpack多入口打包

## 目录说明
```
├── README.md     
├── node_modules
├── package.json
├── src                 #源目录
├── tsconfig.json 
├── webgl-examples.rar  #webgljs源码

```
# Development
```
git clone https://github.com/liseen315/webglstudy.git
cd webglstudy
npm install --registry=https://registry.npm.taobao.org
npm run start
```
